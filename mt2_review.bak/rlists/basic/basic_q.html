We will be using the following implementation of mutable Rlists. NOTE: this implementation may or may NOT be used on the exam. When solving these problems, focus on the process (i.e. how to use first and rest) rather than on the answer.

    empty = None

    class Rlist:
        def __init__(self, first, rest=empty):
            self.first = first
            self.rest = rest

        def __repr__(self):
            if self.rest == empty:
                return 'Rlist({})'.format(self.first)
            return 'Rlist({}, {})'.format(self.first, repr(self.rest))

Basics

1) What type of object can first be (e.g. int, string, function, etc.)? What type of object can rest be?

2) Which of the following are valid Rlist constructors?

  1. Rlist(1, 3)
  2. Rlist(1, None)
  3. Rlist(1, Rlist(4))
  4. Rlist(1)
  5. Rlist()

3) Construct an Rlist that contains the following elements:

    1, 3, lambda x: x, "hello"

4) What is the element at index 2 of the following Rlist?

    Rlist('this', Rlist('is', Rlist('an', Rlist('rlist', Rlist('question')))))

5) What is the length of the Rlist in Q4?
