In this section, we will be practice using tools of functional programming, namely map, filter, reduce.

What would Python print?

    >>> L = [1, 2, 3]
    >>> P = [(1, 2), (3, 4)]
    >>> tuple(map(lambda x: tuple(map(lambda y: x*y, L)), L))
    ______
    >>> tuple(map(lambda x: x*2, P))
    ______
    >>> tuple(map(lambda x, y: (y, x), P))
    ______
    >>> tuple(map(lambda x, y: (x, y), L, L))
    ______
    >>> tuple(map(lambda x, y: x + y, [1, 2, 3], [1, 2]))
    ______
    >>> tuple(map(lambda x, y, z: x + y + z, [1, 2, 3], [1, 2, 3], [1, 2, 3]))
    ______

    >>> tuple(filter(None, L))
    ______

    >>> from functools import reduce
    >>> reduce(lambda x, y: x + y, L, 100)
    ______
    >>> reduce(lambda x, y: (x, y), L)
    ______
    >>> reduce(lambda x, y: y + x, 'hello world!')
    ______

    >>> reduce(lambda x, y: x + y, map(lambda x: x**2, L))
    ______
    >>> tuple(map(reduce, [lambda x, y: x + y]*4, P))
    ______
