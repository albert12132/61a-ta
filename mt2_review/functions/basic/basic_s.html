Code writing

1) Implement map.

    def map(f, seq):
        tup = ()
        for elem in seq:
            tup += (f(elem),)
        return tup

2) Implement filter.

    def filter(pred, seq):
        tup = ()
        for elem in seq:
            if pred(elem):
                tup += (elem,)
        return tup

3) Implement reduce.

    def reduce(combiner, seq):
        total = seq[0]
        for elem in seq[1:]:
            total = combiner(total, elem)
        return total
