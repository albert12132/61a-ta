Basics

1) What condition must be satisfied for the expression a is b to be True?

Answer: a and b must reference the same physical object in memory.

2) What special method determines if a == b is True?

Answer: in each class, the __eq__ determines if two objects are equivalent (but not identical).

  • For user-defined types (i.e. classes), you can implement your own __eq__ method!
  • The default __eq__ for user-defined types behaves the same way as is.

3) For built-in types, if a is b is True, is a == b guaranteed to be True?

Answer: yes. In other words, a built-in object is, by definition, equivalent with itself.

  • However, for user-defined types, you are able to break this property!
