Basics

1) What condition must be satisfied for the expression a is b to be True?

2) What special method determines if a == b is True?

3) For built-in types, if a is b is True, is a == b guaranteed to be True?

What would Python print?

    >>> s = [1, 2, 3, 4]
    >>> s == [1, 2, 3, 4]
    ______
    >>> s is [1, 2, 3, 4]
    ______
    >>> [1, 2, 3, 4] is [1, 2, 3, 4]
    ______
    >>> a = s
    >>> a is s
    ______
    >>> a[1:] is s[1:]
    ______

    >>> s = (1, 2, 3, 4)
    >>> s is (1, 2, 3, 4)
    ______
    >>> s == [1, 2, 3, 4]
    ______

    >>> 'hello' == 'hello'
    ______
    >>> 'hello' is 'hello'
    ______
