Basics

1) Define each of the following terms:

  • entry/datum
  • node
  • child
  • parent
  • leaf
  • forest
  • binary tree

2) Which of the following are valid BinaryTree constructors (based on the implementation provided in the tree.py file)?

  1. Tree()
  2. Tree(3)
  3. Tree(5, Tree(1), Tree(5))
  4. Tree(4, Tree(2))
  5. Tree(2, Tree(2, 4, 5))

3) Draw a graphical representation of the following tree:

    Tree(25, 
         Tree(14,
              Tree(9),
              Tree(20)),
         Tree(30,
              Tree(27)))
  1. What number is contained in the root of this tree?
  2. Which numbers are contained in leaves?
  3. How many children does the node containing 14 have?
