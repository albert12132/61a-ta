Code writing

1) Implement reverse.

    def reverse(L):
        for i in range(len(L)//2):
            L[i], L[-i-1] = L[-i-1], L[i]

2) Implement map_mut.

    def map_mut(f, L):
        for i in range(len(L)):
            L[i] = f(L[i])
