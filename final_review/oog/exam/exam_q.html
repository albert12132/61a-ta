Orders of Growth

For the following questions, find the time complexity of each function in big-Theta (θ) notation.

Q1

Find the time complexity of main.

    def helper(x):
        for i in x:
            print(i)

    def main(n):
        if n == 2:
            return 0
        else:
            return helper(n-1) + helper(n-2)

Q2

Find the time complexity of bar.

    def foo(x):
        for i in range(x):
            for j in range(x):
                print(x)

    def bar(n):
        while n > 0:
            foo(100000)
            n -= 1

Q3

    def subsets(n):
        if n == 0:
            return [[]]
        else:
            result = subsets(n-1)
            for subset in result[:]:
                result.append([n] + subset)
            return result
