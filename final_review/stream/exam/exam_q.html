Conceptual

Q1

Consider the following function:

    def stream1():
        def compute_rest():
            return add_streams(stream1(), stream1().rest)
        return Stream(0, lambda: Stream(1, compute_rest))

List the first 5 elements of the Stream that is returned by stream1().

Q2

Consider the following function:

    def stream2():
        def compute_rest():
            return add_streams(integer_stream(), stream2())
        return Stream(0, compute_rest)

List the first 5 elements of the Stream that is returned by stream2().
